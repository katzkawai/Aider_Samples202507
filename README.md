# ポートフォリオサイト

モダンでカラフルなポートフォリオサイトです。HTML、CSS、JavaScriptを使用して作成されています。

## 特徴

- モダンなデザインとカラースキーム
- インタラクティブな要素とアニメーション
- レスポンシブデザイン
- 3つの主要ページ（ホーム、当サイトについて、お問い合わせ）

## 技術スタック

- HTML5
- CSS3 (Flexbox, アニメーション, 変数)
- JavaScript (ES6+)

## インストール

1. リポジトリをクローン:
```bash
git clone https://github.com/yourusername/your-repo.git
```

2. ブラウザでindex.htmlを開く:
```bash

```
